Fact sheet:

My application is a web based and was written in php. It consists of four pages, the home page, the menu page, the order page and the admin page. The user starts on the home page, from there they are prompted to view the menu page, they can navigate to any of the other pages using the navbar on the top of the screen. From the menu screen the user can select menu items and add them to their order. They can then navigate to the order screen to view their order. From the order screen they can view the total cost of their order, check out their order or cancel their order. Checking out their order removes the quantity of items they selected from the stock. Cancelling their order returns the items in their order into stock. From the admin screen the user can edit existing menu items, add new menu items, and view open orders.
